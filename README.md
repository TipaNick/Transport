## Общее
Консольное приложение на С++, которое принимает на вход тип(типы) транспорта и выводит на консоль его параметры.
## Запуск на Windows
1. Запустить скрипт *build.bat*
2. Запустить программу с использованием параметров, например: ./Transport.exe 1 2
## Запуск на Linux
1. Запустить скрипт *build.sh*
2. Запустить программу с использованием параметров, например: ./Transport 1 2
